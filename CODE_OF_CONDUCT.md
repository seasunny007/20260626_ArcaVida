# 行为准则 / Codigo de conducta / Code of Conduct

本文档提供中文、西班牙语和英文说明；如三种文本存在不一致，以中文文本为准。

Este documento ofrece informacion en chino, espanol e ingles. Si existe alguna inconsistencia entre los tres textos, prevalecera el texto chino.

This document provides Chinese, Spanish, and English information. If there is any inconsistency among the three texts, the Chinese text controls.

ArcaVida 贡献者应让项目保持有用、尊重和安全，尤其要考虑多语言救援协调流程中的敏感场景。

Las personas que contribuyen a ArcaVida deben mantener el proyecto util, respetuoso y seguro, especialmente por la sensibilidad de los flujos de coordinacion de ayuda multilingue.

ArcaVida contributors are expected to keep the project useful, respectful, and safe for people building tools for multilingual relief coordination workflows.

## 期望行为 / Comportamiento esperado / Expected Behavior

- 在讨论、审查和 Issue 分流中保持尊重和务实。
- Mantener respeto y sentido practico en discusiones, revisiones y triaje de issues.
- Be respectful and practical in discussion, review, and issue triage.
- 假设善意，同时清楚说明风险和证据。
- Presumir buena fe, explicando con claridad riesgos y evidencias.
- Assume good intent while staying clear about risks and evidence.
- 优先保护受影响人员的隐私、尊严和安全。
- Priorizar la privacidad, dignidad y seguridad de las personas afectadas.
- Prioritize the privacy, dignity, and safety of affected people.
- 给出和接收具体的技术反馈。
- Dar y recibir comentarios tecnicos concretos.
- Give and receive technical feedback with specifics.
- 适当承认社区知识和一线经验。
- Reconocer adecuadamente el conocimiento comunitario y la experiencia de campo.
- Credit community knowledge and field expertise appropriately.

## 不可接受行为 / Comportamiento inaceptable / Unacceptable Behavior

- 骚扰、威胁、侮辱或歧视性语言。
- Acoso, amenazas, insultos o lenguaje discriminatorio.
- Harassment, threats, insults, or discriminatory language.
- 未经同意发布私人信息、凭证或个人数据。
- Publicar informacion privada, credenciales o datos personales sin consentimiento.
- Publishing private information, credentials, or personal data without consent.
- 鼓励不安全的操作使用，包括未经核实的升级、派遣或现场决策。
- Fomentar usos operativos inseguros, incluidos escalamiento, despacho o decisiones de campo sin verificacion.
- Encouraging unsafe operational use, including unverified escalation, dispatch, or field decisions.
- 反复干扰技术或安全讨论。
- Desviar repetidamente discusiones tecnicas o de seguridad.
- Repeatedly derailing technical or safety discussions.

## 报告 / Reporte / Reporting

如果发现违反行为准则的情况，尽可能先通过私有维护者联系方式处理，再公开升级。如果没有私有联系渠道，可在 GitHub Issue 中使用最少必要信息请求维护者关注。不要在公开报告中包含私人个人数据。

Si observas una infraccion de este codigo de conducta, intenta usar primero una via privada con mantenedores antes de escalar publicamente. Si no hay canal privado, abre un issue con la informacion minima necesaria para pedir atencion del mantenedor. No incluyas datos personales privados en reportes publicos.

If you see behavior that violates this code of conduct, open a private maintainer contact path before public escalation whenever possible. If no private contact is available, open a GitHub issue with only the minimum information needed to request maintainer attention. Do not include private personal data in public reports.

维护者可以移除内容、关闭讨论或限制参与，以保护项目和社区。

Los mantenedores pueden eliminar contenido, cerrar discusiones o limitar la participacion para proteger el proyecto y la comunidad.

Maintainers may remove content, close discussions, or restrict participation to protect the project and community.
